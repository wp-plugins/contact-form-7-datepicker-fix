Add the jQuery Datepicker functionality to your Contact Form 7

== Description ==

You are using the Contact Forms 7 Plugin and are angy about the none IE > 7 Support?
No Problem. With this Plugin u will have the normal Datepicker from jQuery in your Internet Explorers.

If you have Questions or Feature Request, feel free and contact me.

== Installation ==

1. Upload `cf7-datepicker-ie-fixer.zip` to the `/wp-content/plugins/` directory
2. Extract `cf7-datepicker-ie-fixer`
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Preview Form
5. Edit your css file and overwrite Plugin settings
